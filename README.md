# 생활비연구소

**정부지원금 · 생활비 계산 플랫폼**

## 프로젝트 소개

생활비 절약과 정부지원금 정보를 한곳에서 제공하는 웹서비스입니다.

### 주요 기능

- 지원금 찾기
- 전기요금 계산기
- 생활비 절약 정보
- 정부지원금 상세 안내
- AI 추천 시스템(개발 예정)

## 기술 스택

- React
- Vite
- React Router
- GitHub
- Vercel
